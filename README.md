## 🚀 Quick Project Summary

**SkillPath AI** is a real-world, AI-driven web application that helps students:
- Set a learning goal and assess their current skill level
- Get a **personalized AI-generated learning roadmap**
- Access **AI-powered doubt support** (chat assistant)
- Discover **project ideas** matching their level
- **Track progress** through a visual dashboard
- Explore curated **resources** managed by admins

**Stack:** MongoDB · Express.js · React.js · Node.js · Groq AI API
